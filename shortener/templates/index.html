Welcome to a simple URL Shortener!

The purpose of this app is to easily access your favorite sites on the web! The navigation bar should be helpful to execute whatever you command but you intend. However if you want to strictly use URL paths to navigate the site, it has the following functionality:

  • To create a short link from a website use "/shorten/<longURL>", this will show you the short code to type in for the future
  • To use that short code for easy access use "/<shortURL>"
  • To see the 100 most recently created links between short and long URL's use "/recent/"
  • To see the top 10 most visited short URLs use "/top/"
  • To see the amont of times a cetain short URL has been used use "/count/<shortURL>"

NOTE: The website is sensitive to backslashes ('/'), so follow the pattern described above for your intended use.

To use a shortened URL as a redirection to the associated longer URL, in the URL in this browser after the site name, enter a "/" followed by the shortened URL you want to visit!
